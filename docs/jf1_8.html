🌟 The Power of the "Smart Guess" (Estimation)

Estimation means making a smart guess about an answer without needing the exact number.
we can estimate when Counting takes too long, We need a quick answer, We want to predict something.

🎯 Why Estimation is Important

Estimation is not random guessing. It is a guess that is close to the real answer.

💻 With Estimation

• Make Us Think faster
• Solve Problems Quickly
• Can Predict Result

⏱️ Without Estimation

• Robots may crash
• Time to programs become slow
• Movements become inaccurate



🧠 Easy Examples of Estimation

Programmers use estimation to plan, predict, and make decision to solve problem faster

🤖 What is a Color Sensor?

A color sensor is a tool that helps a robot see and recognize colors.
It works like the robot’s eyes 👀 and help robot to make decision depend on the color sensed.

🎯 Key Points to Remember

A color sensor helps robots detect colors.Robots use color
as information to make decisions. Coding tells the robot what to do when colors are detected.

🚦 Traffic Light System

# Color sensor using traffic light system
# Red to stop, Yellow to slow down, & green to move.

if color sensor sense 🔴,
  stop()
elif color sensor sense 🟡,
  slow()
elif color sensor sense 🟢,
  move()

Output : depend on color sensed

🥑 Fruit Selector

# Use color sensor to differentiate fruit
# Each fruit represented according to its color

if color sensor sense 🟣,
  grapes()
elif color sensor sense 🟢,
  Avocado()
elif color sensor sense 🔵,
  blueberry()

Output : You need 2.5 Avocado
